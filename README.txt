ReadMe - Demo Slot (Nan303 style)
==================================

File ini adalah versi demo yang **AMAN** dan **NON-MONETARY**. Untuk menampilkan background Doraemon seperti yang diminta, ikuti langkah berikut:

1. Ganti file 'bg-placeholder.jpg' dengan gambar Doraemon yang kamu miliki dan beri nama file 'doraemon.jpg' (pastikan kamu memiliki hak menggunakan gambar tersebut).
2. Upload seluruh isi folder ini ke repository GitHub (root). Jangan letakkan di dalam subfolder.
3. Aktifkan GitHub Pages di Settings -> Pages -> Branch: main, Folder: /(root).
4. Buka https://<username>.github.io/<repo> atau jika repo diganti ke username.github.io, buka https://<username>.github.io/

Catatan hukum: Pastikan gambar Doraemon yang kamu pakai tidak melanggar hak cipta. Jika tidak yakin, gunakan gambar free-to-use atau minta izin pemilik.
